 Environment requirements to start up TRUST 


To set up the login named for example LOGIN_TRUST,
and to use the TRUST versionX.Y already installed,
follow the next instructions: 

1) To update the environment variable PATH :

a) With ksh environment, in $HOME/.profile :

export PATH=$LOGIN_TRUST/VersionX.Y/TRUST/bin:$PATH

b) With csh environment, in $HOME/.cshrc :

setenv PATH $LOGIN_TRUST/VersionX.Y/TRUST/bin:$PATH

2) The new environment will be updated :

a) With ksh :

$ . $HOME/.profile

b) With csh :

% source $HOME/.cshrc

3) To run TRUST, just type :

TRUST

The TRUST main window will appear, through the Netscape browser.
