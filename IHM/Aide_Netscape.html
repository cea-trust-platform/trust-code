Netscape 3.x Configuration 

How to configure your HOME page from TRUST page:
Menu Options
Menu General preferences
Click on Appearance button
In Home Page Location, enter file address: $HOME/HTML/HOME.html
And valid OK.

To get and/or send mail via Netscape:
Menu Options
Menu Mail and News preference
Click on Servers button
Choose Built in Movemail and valid it.
Then click on  Identity button
Give your name in Identity (option)
Give your e-mail address in  Email Adress
If the directory $HOME/nsmail does not exist, create it with the Unix command: 
$ mkdir $HOME/nsmail

For the LATA's users: how to access to the internal network CEA sites :
Menu Options
Menu Network preferences
Click on  Proxies button
Then  Manual proxy configuration, choose View
Full the following items :
FTP Proxy:      fire-dtp        Port:14000
HTTP Proxy:     fire-dtp        Port:14000
Then valid OK twice.

Now, you can access to the following addresse:

The CEA site : http://www.cea.fr
The SYSCO site: http://www-drn2.cea.f
The TRUST site: ftp://ftp.cea.fr/private/TRUST/mddz87J/index.html
